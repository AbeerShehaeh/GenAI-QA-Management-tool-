# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1Du-9CxT21D4RmlnPc3D4gvZWeSwcOOQs?fullscreenApplet=true

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`


## Project Overview

GenAi_QA Management Tool is a Proof-of-Concept AI-powered tool for
extracting software requirements, generating test cases, and providing
full traceability and QA reporting. Built with Google Build.

## Features:
- Extract requirements from PDF/DOCX files automatically
- Generate AI-driven test cases
- Traceability matrix for requirements-to-tests
- QA analytics dashboard
